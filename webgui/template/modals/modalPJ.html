Assistance technique Groupe Solocal

En charge d’intégration/ingénierie système sur les projets des filiales Pages Jaunes, ainsi que d’outillage et du support N3 sur les infrastructures. Gestion des environnements de tous les projets (filiales, projets d'infrastructure, outillages) avec Fabric/Ansible et Gitlab.
  • Projet Référentiel de Contenus et de Clients (RCC) :
    • Serveurs applicatifs avec framework Nuxeo et Apache
    • Base de données MongoDB en ReplicaSet
    • Cluster Elasticsearch 5 noeuds
    • Intégration d’une application tierce utilisant NodeJS (v4 et v6) + Nginx
    • Mise en place de tests fonctionnels pour le projet lancés à chaque commit avec Gitlab-CI + gestion de branches GIT
    • Ajout d’une base de données InfluxDB pour envoi des métriques du framework Nuxeo vers une infra Grafana + réalisation de quelques dashboards Grafana
    • Configuration des middlewares pour envoyer les logs vers un serveur Graylog
    • Configuration de streams par environnement dans Graylog

  • Projets RH :
    • 2 serveurs redondés Tomcat/MySQL
    • 2 serveurs Talend/Karaf (configuration/sécurisation de Karaf)
    • 2 serveurs MySQL Master/Master pour Karaf

  • Montée en compétence sur le système d’installation des serveurs :
    • Ajout d’un nouveau serveur PXE pour gérer les nouvelles installations de serveurs (Ubuntu 14.04, 16.04, RHEL7 …), avec gestion du boot UEFI dans le partitionnement
    • Maintenance du serveur PXE existant (installations Ubuntu 12.04, CentOS6/7)
    • Ajout de paramètres dans le menu PXE pour automatiser le boot Gparted
