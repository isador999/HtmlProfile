Cellule intégration système SII

En charge de missions dans la cellule intégration systèmes SII pour de multiples clients/projets
  • Plateformes de qualification / Sandbox (projet PAB, OrangeSoft) :
    • Serveurs applicatifs Tomcat
    • Base de données Couchbase avec plugin N1QL
    • Configuration RabbitMQ
    • Optimisation/Sécurisation du SSH (clés DSA + keychain)
    • Load-balancers utilisant du Rewriting et certificats SSL (Haproxy)
    • Ansible (playbooks, facts, rôles … )
    • Supervision et Capacity Planning (Nagios + Nrpe / Munin)
    • Remontée de logs (outils Elasticsearch-Logstash-Kibana)
    • Déploiement des nouvelles versions (archives WAR)

  • Tierce-maintenance applicative (projet SAFE@SCS, OBS) :
    • Développement de contrôles à base d’un langage libre similaire au C (HAWK), permettant de vérifier des configurations d’équipement (Juniper, BigIP … )
    • Utilisation d’expressions régulières dans ce langage spécifique.
    • Réalisation de scripts shell faisant appel aux contrôles compilés.

  • Audit de sécurité et Hardening système :
    • Autoformation aux tests d’intrusion (WebGoat, rootme).
    • Audit applicatif en mode BlackBox d’une application web SII.
    • Analyse et hardening système sur différents serveurs/services (Ubuntu/Debian, Apache, Mysql, PHP)

  • Mise en place d’un serveur de packaging :
    • Structure de scripts shell permettant de créer des projets, récupérer des sources via GIT, générer des packages de type DEB ou TARGZ,…
    • Interface web avec Django 1.7

  • Etudes et Proof Of Concept sur quelques sujets :
    • Docker (Boot2Docker, génération d’images depuis des Dockerfiles, lancement de conteneurs avec SystemD ou Docker-compose, … )
    • Elasticsearch-Logstash-Kibana couplé à Logstash-Forwarder/FileBeat (ajout de champs, enregistrement dynamique d’indexes Elasticsearch … )
    • Dualboot Windows / Ubuntu (PXE, Preseed, Gparted… )
    • Installation d’un PoC d’Openstack Juno
